# QA-Gmail-Log-In

## About
This project contains basic UI automation tests for Gmail login functionality using Selenium WebDriver and Java.

It was created as part of my early learning in test automation.

## Tech Stack
- Java
- Selenium WebDriver
- JUnit

## Test Coverage
- Login with valid credentials
- Login with invalid credentials

## Notes
- This project demonstrates basic WebDriver usage:
  - locating elements
  - sending input
  - clicking buttons
  - simple waits

- Hardcoded credentials are used for learning purposes only.

## How to Run
1. Install Java and Maven
2. Download ChromeDriver and update the path in the test
3. Run tests via IDE (IntelliJ / Eclipse) or using Maven

## Status
Archived learning project.  
Currently transitioning to Playwright (JavaScript) for modern test automation.
